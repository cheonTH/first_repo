# Mark Down 문법
## 제목
### 주제
#### 본문
##### 주석
- 가스를 연다.
- 냄비에 물을 넣는다.
- 스프와 면을 넣는다.
  - 스프를 먼저 넣는다.
  - 면을 넣는다.
- 면이 풀어질 때 까지 팔팔 끓인다.

### 네이버 접속하는 법
1. 컴퓨터를 켠다.
2. 브라우저를 연다.
3. 네이버를 접속한다.
